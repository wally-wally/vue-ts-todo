# :page_with_curl: TODO LIST with Vue.js & Typescript

<br>

## :one: Overview

- `Vue.js` 프레임워크에서 `Typescript` 언어를 이용해서 만든 TODO LIST 프로젝트

- `Vue.js` + `Typescript` 환경에서 기본적인 컴포넌트 구성과 `vue-router`, `vuex`, `axios`를 사용하는 방법을 익히는 것을 주목적으로 함
- 추후 스타일링과 다양한 기능을 추가할 예정

<br>

## :two: Getting Started

- Clone project & Install library

```bash
git clone https://github.com/wally-wally/vue-ts-todo.git
cd vue-ts-todo
npm install
npm run serve
```

- 서버 실행 후 `http://localhost:8080`에 접속하면 된다.
